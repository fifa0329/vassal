Expression Builder



The Expression Builder is access by clicking on the Calculator icon located to the immediate right of each Expression field. It assists with building valid BeanShell Expressions,

As you type the green tick or red cross indicate whether the syntax of the current expression is valid or not.

The Insert button displays a pop-up menu showing the components of the BeanShell Language and the valid Properties available in the current counter or component.
